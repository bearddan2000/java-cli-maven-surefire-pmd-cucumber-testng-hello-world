# java-cli-maven-surefire-pmd-cucumber-testng-hello-world

## Description
A POC for maven app using testNg
and cucumber framework with
pmd and surefire plugins.

## Tech stack
- java
- maven
  - testNg
  - surefire
  - cucumber
  - pmd

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`
- pmd report found at bin/target/site

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
